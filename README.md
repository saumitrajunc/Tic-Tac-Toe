# Tic-Tac-Toe with Minimax AI

Python Tic-Tac-Toe with an unbeatable AI. Built step-by-step as a beginner CS project.

## Run

In terminal:

```bash
python3 main.py        # graphical UI (default)
python3 main.py --cli  # terminal mode
```

## Files

- `board.py` — board state and game rules
- `ai.py` — Minimax AI
- `UI.py` — graphical interface
- `main.py` — entry point

You play as **X**. The AI plays as **O**
